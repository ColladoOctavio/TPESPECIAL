Trabajo Práctico Especial – Siniestros Viales en Autopistas de Buenos Aires (AUSA)
Repositorio: https://github.com/ColladoOctavio/TPESPECIAL

Este proyecto contiene el análisis exploratorio, estadístico y la visualización de datos sobre siniestros viales ocurridos en autopistas de la Ciudad de Buenos Aires administradas por AUSA (año 2022).

REQUISITOS PREVIOS (instalar antes de comenzar)

Software necesario:

Visual Studio Code: https://code.visualstudio.com/

Python 3.10 o superior: https://www.python.org/downloads/

Git: https://git-scm.com/downloads

Jupyter Lab o Jupyter Notebook: https://jupyterlab.readthedocs.io/en/stable/getting_started/installation.html

CÓMO CLONAR Y EJECUTAR EL PROYECTO

Crear una carpeta donde quieras guardar el proyecto (opcional).

Abrir Visual Studio Code y abrir una terminal desde:
Terminal → New Terminal.

Navegar hasta la carpeta donde querés clonar el repositorio, usando:
cd ruta/a/tu/carpeta

Clonar el repositorio:
git clone https://github.com/ColladoOctavio/TPESPECIAL

Entrar a la carpeta del proyecto:
cd TPESPECIAL

CREAR Y ACTIVAR UN ENTORNO VIRTUAL

Crear el entorno virtual:
python -m venv venv

Activar el entorno:

En Windows:
venv\Scripts\activate

En Linux / MacOS:
source venv/bin/activate

INSTALAR DEPENDENCIAS DEL PROYECTO

Instalar las librerías necesarias:
pip install -r requirements.txt

(Asegurate de estar dentro de la carpeta del proyecto antes de ejecutar este comando).

EJECUTAR LA NOTEBOOK

Podés ejecutar la notebook desde Visual Studio Code usando la interfaz, o bien iniciar Jupyter manualmente.

Para iniciar Jupyter desde la terminal:
jupyter notebook
o
jupyter lab

Luego abrir el archivo:
TPESPECIAL.ipynb
